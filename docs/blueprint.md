# **App Name**: Proximity Network

## Core Features:

- User Authentication: Sign-up/Login pages with secure password handling.
- Profile Management: Profile pages to manage personal and professional information, including photo uploads and LinkedIn URL.
- Map View: Display other logged-in users on a map, showing a snippet of profile information, with navigation to their profiles.
- User List: Display all registered users in a list, sortable and searchable by profession, experience, location and online status.
- Public Profile: Contact icons (LinkedIn, email, message) for easy interaction.
- In-App Messaging: Real-time chat with status indicators and notifications.
- Smart Suggestions: AI-powered tool for generating suggestions for expanding one's professional profile, or icebreaker conversation starters.

## Style Guidelines:

- Primary color: Navy blue (#3B5998), reflecting professionalism and trustworthiness.
- Background color: Light gray (#F0F2F5), providing a clean and modern feel.
- Accent color: Sky blue (#5BC0DE) for highlights and interactive elements.
- Body text and headlines: 'Inter', sans-serif font, for a modern, clean and neutral aesthetic.
- Clean and professional icons for navigation and actions.
- Clean and organized layout with clear separation of sections.
- Subtle animations for user interactions, such as loading screens and button hovers.
